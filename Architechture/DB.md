## PostgreSQL (Primary Relational Data):

- Guild configurations
- User profiles
- Moderation logs
- Command history
- Leveling/XP data
- Relationships requiring transactions

## Redis (Caching & Real-time):

- Active guild count
- User session data
- Rate limit tracking
- Leaderboard cache
- Command cooldowns
- Temporary data (expires automatically)

## MongoDB (Analytics & Semi-structured):

- Server statistics
- User behavior analytics
- AI conversation logs
- Event audit trails
- Flexible schema data
